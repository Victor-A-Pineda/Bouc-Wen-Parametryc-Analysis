# Bouc-Wen-Parametryc-Analysis

Structural systems subjected to intense cyclic loads, such as those from earthquakes, exhibit nonlinear and inelastic behavior. Analyzing these systems is complicated because they behave as systems with memory due to the appearance of a hysteresis loop in the force-deformation relationship. The Bouc-Wen model allows for the convenient determination of the dynamic response of these systems by solving a system of nonlinear differential equations. However, the results depend on four parameters that are typically obtained through trial and error. In this study, a parametric analysis of the model was developed, considering different ranges for the parameters defined through random sampling. Furthermore, an interactive graphical interface was developed to facilitate the implementation of the model and to allow the user to observe and examine the results obtained by changing the parameters and seismic excitations.
